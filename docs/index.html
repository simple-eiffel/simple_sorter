simple_sorter

Modern, ergonomic sorting for Eiffel collections

Features

Agent-Based Comparison

No comparator classes needed. Just pass an agent.

Multiple Algorithms

Introsort, merge sort, heap sort, insertion sort.

Stable Sorting

Preserve order of equal elements when needed.

Full DBC

Design by Contract throughout - 37 contract clauses.

SCOOP Compatible

Ready for concurrent Eiffel applications.

Void-Safe

100% void safety with void_safety="all".

Quick Start

local
    sorter: SIMPLE_SORTER [PERSON]
    people: ARRAYED_LIST [PERSON]
do
    create sorter.make

    -- Sort by name (ascending)
    sorter.sort_by (people, agent {PERSON}.name)

    -- Sort by age (descending)
    sorter.sort_by_descending (people, agent {PERSON}.age)

    -- Stable sort preserves order of equal elements
    sorter.sort_by_stable (people, agent {PERSON}.department)
end
